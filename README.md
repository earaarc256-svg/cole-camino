# El cole marca nuestro camino

Web estática para presentar la memoria justificativa del proyecto educativo **“El cole marca nuestro camino”**.

## Estructura

- `index.html`: contenido y secciones de la memoria.
- `styles.css`: diseño visual responsive.
- `script.js`: navegación móvil, animaciones, filtros, pestañas y contadores.

## Ejecutar en local

```bash
python3 -m http.server 4173
```

Después abre `http://localhost:4173`.
